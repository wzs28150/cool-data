# 模板

模板组件,定义了可视化大屏的一些场景的模块摆放形式

# 公共设置

参数说明类型是否必要默认值
background背景颜色Stringfalse-
width大屏宽度Numberfalse1920
height大屏高度Numberfalse1080
isScale是否缩放, 设置为 true 页面会随着比例缩放, 设置为 false 时页面不够指定宽高时显示滚动条Booleanfalsetrue

# 设置

参数说明类型是否必要默认值
leftSideWidth左侧区域宽度, 如设置数字则默认单位 px`StringNumber`false
rightSideWidth右侧区域宽度, 如设置数字则默认单位 px`StringNumber`false
titleHeight标题高度, 如设置数字则默认单位 px`StringNumber`false
immerse设置是否沉浸, 如设置true则为沉浸模式, 主区域铺面整屏, 如设置false则为非沉浸模式, 主区域为区块`StringNumber`false

# 插槽

名称说明
layout-title头部区域
left左侧区域
right右侧区域
main主区域
