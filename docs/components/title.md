# 标题
> 标题组件, svg绘制 总结了一些大屏顶部大标题的样子

### 公共设置

| 参数            | 说明                                               | 类型     | 是否必要 | 默认值 |
| --------------- | -------------------------------------------------- | -------- | -------- | ------ |
| color           | 颜色,可设置颜色关键字、十六进制色、RGB 及 RGBA     | String[] | false    | -      |
| title           | 标题文字    | String | true    | -      |
