# 装饰器

## 装饰器1

## 示例：

<ClientOnly>
<div class="code-show" style="height: 300px;">
  <decoration-1 />
</div>
</ClientOnly>

#### 代码：

```html
<template>
  <decoration-1 />
</template>
```

#### 设置：

| 参数 | 说明 | 类型   | 是否必要 | 默认值 |
| ---- | ---- | ------ | -------- | ------ |
| color   | 颜色设置[边框颜色，背景颜色], 背景颜色可设置为transparent透明  | String[] | false    | -      |
