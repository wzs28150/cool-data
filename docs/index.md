## 快速开始

> 在 main.js 中引入组件库

```javascript
// 全部引入
import 'cool-data/dist/css/index.css'
import CoolData from "cool-data";
Vue.use(CoolData);

// 按需引入
import 'cool-data/dist/css/xxx.css'
import { xxx } from "cool-data";
Vue.use(xxx);
```
